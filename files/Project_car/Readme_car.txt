Project: Car Maintenance Tracker

Описание:
Этот проект состоит из двух основных компонентов: класса Car и набора тестов для проверки его функциональности. Класс Car предназначен для отслеживания статуса обслуживания автомобилей, включая их марку, модель, год выпуска, пробег, дату последнего обслуживания и зарегистрированные проблемы. Тесты написаны с использованием pytest и обеспечивают правильную работу методов класса Car, включая логирование соответствующей информации.

Файлы:
1. cars.py: Этот файл содержит реализацию класса Car и функцию для загрузки данных об автомобилях из текстового файла 'cars.json'.
2. Pytest_cars.py: Этот файл содержит тесты для класса Car с использованием pytest. Включает фикстуры для корректных и некорректных данных об автомобилях и проверяет функциональность и логирование класса Car.

Использование:
- Запустите тесты, чтобы убедиться, что класс Car работает правильно.
- Используйте класс Car для создания объектов автомобилей и отслеживания их статуса обслуживания.
- Используйте функцию load_cars_from_file для загрузки данных об автомобилях из текстового файла.

Требования:
- Python 3.6+
- pytest
- Logging (логирование)

Установка:
1. Убедитесь, что у вас установлен Python версии 3.6 или новее.
2. Установите необходимые пакеты с помощью команды: pip install -r requirements.txt
3. Запустите тесты с помощью команды: pytest Pytest_cars.py
